# 2.对于refresh函数频繁调用的问题进行处理

  • 进行debounce防抖
  • 防抖函数起作用的过程
    • 如果我们执行refresh,那么refresh函数会被执行30次
    • 可以将refresh函数传入debounce,生成一个新的函数
    • 之后在调用频繁时候就会调用新的函数
    • 而新生成的函数就不会频繁调用,如果下一次执行的很快,那么上一次就会被取消掉
debounce(func,delay){
        let timer=null
        return function(...args){
            if(timer) clearTimeout(timer)//如果timer不为空就清除timer
          timer=setTimeout(()=>{
            func.apply(this, args);
          },delay)
        }
      },

# 3.tab-control的吸顶效果的实现

# 3.1 获取tab-control这一栏的offsetTop属性

问题:

  • 1.tab-control组件是没有offsetTop属性的

    • 能通过this.$refs.ref名.$el.offsetTop可以得到对应ref值组件里面元素的offsetTop(偏置顶部的距离)

  • 2.得到的offsetTop值是不准确的(图片未加载完成得到的offsetTop)

    • 通过监听对应的图片加载事件,加载事件完成再得出的offsetTop是相对准确的

  • 3.得到的offsetTop保存的data中,设置tab-control一个:class属性。监听滚动位置，当位置>=offsetTop的时候，开启class属性，设置对应的CSS来实现吸顶效果。

    :class="{fixed:isShowTabContro}"
isShowTabContro:false //默认是false
